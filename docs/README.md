# Omnipath v2 Documentation

This directory contains the complete documentation for the Omnipath v2 platform.

## Sections

- **[User Guide](./user_guide/README.md):** For end-users of the Omnipath platform. Explains how to create tenants, manage agents, define missions, and interpret results.
- **[Developer Guide](./developer_guide/README.md):** For developers contributing to the Omnipath codebase. Covers architecture, local setup, testing, and contribution guidelines.
- **[Operations Runbooks](./ops_runbooks/README.md):** For SREs and operators managing Omnipath in production. Contains procedures for deployment, monitoring, alerting, and incident response.
- **[API Reference](./api/README.md):** Detailed OpenAPI specification and Postman collection for the Omnipath v2 REST API.

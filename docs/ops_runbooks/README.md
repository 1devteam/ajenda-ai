# Operations Runbooks

This section contains runbooks for Site Reliability Engineers (SREs) and operators responsible for the Omnipath v2 production environment.

## Contents

1.  **[Deployment](./01-deployment.md):** Step-by-step guide for deploying Omnipath v2 to a Kubernetes cluster.
2.  **[Monitoring & Alerting](./02-monitoring.md):** Overview of the monitoring stack (Prometheus, Grafana, Loki) and how to interpret key dashboards and alerts.
3.  **[Incident Response](./03-incident-response.md):** Playbooks for responding to common critical alerts (e.g., `OmnipathBackendDown`, `HighErrorRate`).
4.  **[Database Management](./04-database.md):** Procedures for database backups, restores, and migrations.
5.  **[Scaling](./05-scaling.md):** How to scale the application horizontally and vertically in response to load.
6.  **[Troubleshooting](./06-troubleshooting.md):** Common issues and their resolutions.

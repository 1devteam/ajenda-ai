# API Reference

This section provides detailed documentation for the Omnipath v2 REST API.

## OpenAPI Specification

The canonical source of truth for the API is the OpenAPI 3.0 specification.

- **[View in Swagger UI](./openapi.html):** An interactive UI for exploring the API endpoints.
- **[Download OpenAPI JSON](./openapi.json):** The raw OpenAPI specification file.

## Postman Collection

A Postman collection is provided for easy testing and interaction with the API.

- **[Download Postman Collection](./omnipath_v2.postman_collection.json):** Import this file into Postman to get started.
